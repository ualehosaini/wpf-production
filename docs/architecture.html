Overview

A modular WPF desktop application demonstrating end-to-end Track & Trace across a production station. Built on a plugin-based architecture: each station type is a separate assembly implementing IStationPlugin, loaded at runtime. Production data is captured in real-time to a SQLite database and can be exported to CSV for ERP reporting.
